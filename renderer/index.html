Настройка OAuth

Для работы приложения нужны OAuth-учётные данные из Google Cloud Console. Инструкция по настройке

Ошибка access_denied: добавьте свой Google-email в Google Auth Platform → Audience → Test users. Открыть Audience
Обязательные API: включите в Google Cloud Console Google Search Console API и Web Search Indexing API.
